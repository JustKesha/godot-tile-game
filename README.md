# 🕹️ Godot 3D ✦ Interactive Tiles

Modular base for a first-person Godot 3D game with customizable interactive tiles.

This is a rework of my older repository - [godot-interactive-tiles](https://github.com/JustKesha/godot-interactive-tiles)

### Features
- Easily customizable tile visuals & interactions
- Tiles account for the weight applied to them 
- Basic first-person controller

#### Whats new?
- Improved resource-driven architecture
- Convenient race-condition-free solution using the new RPG-style stats system

### Requirements
- Godot 4.6 or higher
